## v1.0.0-alpha.4 - 2025-12-08
### ‚ú® Added
* Interactive CLI for time tracking with Jira and Tempo integration
* Task selection from in-progress issues, search, or manual entry
* Multiple time format support (2h 30m, 2.5h, 150m)
* Configurable shortcuts for quick logging with predefined tasks, labels, and time
* Shortcuts available as subcommands (tasklog daily, tasklog standup) with dynamic help listing
* SQLite local cache for offline time tracking with sync recovery
* Break management with Slack integration for status updates and channel notifications
* Daily summary showing Tempo worklogs as source of truth with sync status indicators
* Sync command to recover and retry failed Jira/Tempo submissions
* Project key filtering for focused task selection
* Configurable work labels with optional label filtering
* jira: allow configuring multiple task statuses to include when fetching tasks (e.g., 'In Progress', 'In Review'). Defaults to 'In Progress' if not specified.
* init: add --update flag to intelligently migrate existing config files by removing deprecated fields and adding new ones as comments
* Added upgrade command to automatically update tasklog binary from GitHub releases with support for stable and pre-release channels
* version: add version subcommand to display detailed build information, and hide upgrade command in non-official builds
### üîÑ Changed
* log: show friendly message when summary is disabled instead of Tempo API error. Summary now only displays when Tempo is enabled in config.
* replaced custom time parsing implementation with go-str2duration library for better maintainability and standards compliance
* Switch SQLite driver to modernc.org/sqlite and enable CGO-free cross compilation
* init: support config schema validation and migration and add --update flag for init command
### üêõ Fixed
* README: updated download instructions with correct binary naming format (without 'v' prefix) and added placeholder template with example
### ‚ö° Performance
* improve prepare release workflow in github
